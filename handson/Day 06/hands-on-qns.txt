~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    10, 26, 51, 87, 136, ... N terms

3. Find Number of Odd Digits in a given number 
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Example:
12345
Odd Digits are 1, 3 and 5 So Number of Odd Digits is 3 
Example:
23456
Odd Digits are 3 and 5 So Number of Odd Digits is 2
